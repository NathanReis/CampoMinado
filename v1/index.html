Campo Minado

Este jogo foi desenvolvido em Javascript e é possível jogar apenas ferramenta de desenvolvimento do navegador.

Aperte F12 ou Ctrl + Shift + I para abri-lá. Caso os atalhos não funcionem, verifique qual o correto para sua máquina e navegador.

Com a ferramenta de desenvolimento aberta, vá na aba "Console".

Caso esteja em um dispositivo móvel ou não queira jogar pelo conselo, click aqui para jogar pela interface.

Comandos

  • Abir uma posição no campo:
    abrirPosicao(x, y)
    x: A linha que deseja abrir.
    y: A coluna que deseja abrir.
  • Marcar uma possível bomba:
    marcarComBandeira(x, y)
    x: A linha que deseja marcar.
    y: A coluna que deseja marcar.
  • Reiniciar:
    reiniciar()
