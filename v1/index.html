Campo Minado

Este jogo foi desenvolvido em Javascript e é possível jogar com a ferramenta de desenvolvimento do navegador ou via interface com um emulador do console.

Aperte F12 ou Ctrl + Shift + I para abri a ferramenta de desenvolvimento. Caso os atalhos não funcionem, verifique qual o correto para sua máquina e navegador.

Com a ferramenta de desenvolimento aberta, vá na aba "Console".

Caso esteja em um dispositivo móvel ou não queira jogar pelo console, click aqui para jogar pela interface.

Comandos

  • Abir uma posição no campo:
    MODO_CONSOLE.abrirPosicao(x, y)
    x: A linha que deseja abrir.
    y: A coluna que deseja abrir.
  • Marcar uma possível bomba:
    MODO_CONSOLE.marcarComBandeira(x, y)
    x: A linha que deseja marcar.
    y: A coluna que deseja marcar.
  • Reiniciar:
    MODO_CONSOLE.reiniciar()
